Copista

This a small Tool to manage your clipboard


Copy Text - From Input Element

To copy data from a input element just add data-copy-origin="#element" attribute to the button.

<button class="btn btn-charlie" data-copy-origin="#txtCopy">Copy</button>

Copy Text - From TextArea Element




Copy Text - From Nowhere

To copy text to clipboard data-copy-text="text" attribute to the button.

<button class="btn btn-charlie" data-copy-text="Some text">Copy</button>



Copy Text - From Others Element

To copy text to clipboard from a html element just include data-copy-origin="#element" attribute to the button.

<button class="btn btn-charlie" data-copy-origin="#element">Copy</button>

Text copied from a p element.




Paste Text

To paste data to a input element just add data-paste-target="#element" attribute to the button.

<button class="btn btn-charlie" data-paste-target="#txtPaste">Paste</button>


Cut Text

To cut data from a input element just add data-cut-origin="#element" attribute to the button.

<button class="btn btn-charlie" data-cut-origin="#txtCut">Cut</button>


Copy and Paste Text

To copy and paste data from a input element to another just add data-copy-origin="#element" and data-paste-target="#element" attributes to the button.

<button class="btn btn-charlie" data-copy-origin="#txtCopy2" data-paste-target="#txtPaste2">Copy and Paste</button>
